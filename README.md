# OAuth Authentication project

.env file content:
```
{
  REACT_APP_OAUTH_CLIENT_ID=clientId
  REACT_APP_OAUTH_AUTH_ENDPOINT=authorize endpoint
  REACT_APP_OAUTH_TOKEN_ENDPOINT=token endpoint
  REACT_APP_OAUTH_REDIRECT_URI=redirect uri
}
```
